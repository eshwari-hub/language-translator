# 🌐 Smart Language Translator

A modern web-based language translation tool built using HTML, CSS, and JavaScript.

## 🚀 Features
- 🌍 Translate text between multiple languages
- 🔊 Text-to-Speech (listen to translation)
- 📋 Copy translated text
- 🔁 Swap languages
- 🎨 Modern aesthetic UI (glassmorphism design)

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript
- Google Translate API (unofficial endpoint)

## ▶️ How to Run
1. Download the project
2. Open `index.html` in your browser
